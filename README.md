# Frontend - Complaint Management System

Professional React frontend for the Complaint Management System with modern UI/UX design.

## 🚀 Tech Stack

- **React** - UI library
- **React Router** - Navigation
- **Axios** - HTTP client
- **Context API** - State management
- **Modern CSS** - Custom design system

## 🎨 Features

- ✨ Modern glassmorphism design
- 🌈 Purple/pink gradient theme
- 💫 Smooth animations and transitions
- 📱 Fully responsive design
- 🔐 JWT authentication
- 👥 Role-based dashboards (USER & SUPPORT)
- 📊 Real-time statistics
- ⭐ 5-star feedback system

## 📦 Installation

```bash
npm install
```

## 🔧 Configuration

Update the API URL in `src/services/api.js` or use the proxy in `package.json`:

```json
"proxy": "http://localhost:5000"
```

## 🚀 Running the App

```bash
# Development mode
npm start

# Build for production
npm run build
```

## 📱 Pages

- **Home** - Landing page with features
- **Login** - User authentication
- **Register** - New user registration
- **Dashboard** - Role-based dashboard with complaints and stats
